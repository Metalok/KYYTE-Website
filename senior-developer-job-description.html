    Senior Java Webstack and Nodejs developer

    We are looking for collaborative and communicative developer who is detail oriented in developing Nodejs application with strong integration knowledge on using OData Rest API’s.

    Job Type : Full Time

    Location : Remote

    Job Description

    We are looking for collaborative and communicative developer who is detail oriented in developing Nodejs application with strong integration knowledge on using OData Rest API’s.

    Design APIs, develop shippable code, documentation, and unit test new features for digital products.


    • Works with fellow API Developers, Team Leads, Architects to deliver features through the creation of re-usable RESTful APIs.
    • Implementation experience in developing applications using Nodejs framework.
    • Perform pair programming, effectively communicate ideas with the team, assist in systems integration, performance testing and product releases
    • Implement policies, roles, data access controls, monitoring events, resolve system and data issues for continuous functioning of APIs

    Qualifications

    • Must have 3 years of API / Server-side development experience in Node.js / Backend JavaScript.
    • Must have 1 year of experience in cloud platforms (SAP BTP, AWS, GCP), developing micro-services, CI/CD solutions (including Docker), message queue systems and background task management.
    • Experience with Kubernetes and Cloud Foundry.
    • Must have 1 year of experience in developing integrated to PostgreSQL DB, designing data models and proficient in querying document stores (preferably PostgreSQL DB or HANA DB).
    • Knowledge in API security frameworks, token management and user access control including OAuth, JWT, Open API, etc.
    • Must demonstrate strong programming skills in JavaScript.
    • Experience working with API Gateway, CDNs, API Performance testing, CI/CD pipelines and monitoring tools.
    • Ability to work in an Agile / SCRUM environment.
    • Self-directed with a start-up/entrepreneur mindset.
    • BE in computer science or similar relevant field
    • Strong writing and communication skills.
