    Senior SAP UI5 developer

    We are looking for collaborative and communicative developer who is detail oriented in developing Fiori UI5 application with strong integration knowledge on using OData Rest API’s.

    Job Type : Full Time

    Location : Remote

    Job Description

    We are looking for collaborative and communicative developer who is detail oriented in developing Fiori UI5 application with strong integration knowledge on using OData Rest API’s.

    Design APIs, documentation, and unit test new features for digital products.


    • Design prototypes and mockups using SAP UI5 framework.
    • Implementation experience in developing applications using Nodejs framework.
    • Co-ordinate with fellow team members in utilizing NodeJs backend services to deploy applications to cloud foundry.
    • He should have a good understanding of front-ended solutions and have basic knowledge of SAP systems.
    • Assist in the integration of front-end UI/UX components with back-end technical and application components to meet business requirements.

    Qualifications

    • Must have 3 years SAPUI5 development & SAP Fiori UI/UX.
    • Must have 1 year of experience in cloud platforms (SAP BTP, AWS, GCP), developing micro-services, CI/CD solutions (including Docker).
    • SAP Fiori Launchpad configuration and application integration.
    • SAP NetWeaver Gateway (OData) Services Development with Basic ABAP.
    • Eclipse development environment with local server testing.
    • HTML5, CSS3, JSON.JQuery, JavaScript.
    • Familiarity with the whole web stack, including protocols and web server optimization techniques.
    • Strong analytical skills and problem-solving aptitude.
